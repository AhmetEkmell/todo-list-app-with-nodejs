## .env dosyası içeriği hakkında

```python
# EXPRESS SERVER CONFIG
PORT=3000                      # Sunucunun çalışacağı port numarası
HOST=localhost                 # Sunucunun çalışacağı host adresi

# SQL SERVER CONFIG
SQL_USER=your_db_user          # MSSQL kullanıcı adı
SQL_PASSWORD=your_db_password  # MSSQL kullanıcı şifresi
SQL_DATABASE=your_db_name      # MSSQL veritabanı adı
SQL_SERVER=your_db_server      # MSSQL sunucu adresi
SQL_ENCRYPT=false              # MSSQL bağlantısı için şifreleme (true/false)
```
